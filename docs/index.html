Willkommen bei Surveyhawk

Surveyhawk ist Ihr Werkzeug zur Analyse und Verbesserung der Gesundheit von Code-Repositories.

Features

Surveyhawk bietet eine Vielzahl von Funktionen, darunter:

  • Analyse von Repository-Metriken
  • Automatisierte Visualisierung der Ergebnisse
  • Verbesserung der Codequalität
